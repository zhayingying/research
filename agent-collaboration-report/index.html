Part 3 / Protocol & SDK Layers

协议与 SDK 分层机制

智能体到智能体协议（Agent-to-Agent Protocol, A2A）解决 Agent 应用互通；模型上下文协议（Model Context Protocol, MCP）解决工具和上下文连接；智能体 SDK（Agent SDK）解决运行。

A2A 跨 Agent 应用互通

Agent Card、task、message、artifact。

MCP 工具上下文连接

tool、resource、prompt、server/client。

SDK 单 Agent 运行对象

model、tool、handoff、guardrail、trace。
