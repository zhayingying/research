## 一、调研目标

为一位 23 岁的早期员工建立关于"公司、股权、融资、退出"的完整认知框架。这份认知有双重用途：

1. [ ] **当下视角** — 理解自己作为早期核心员工持有的股权 / 期权意味着什么、什么时候能变现、可能的路径有哪些
2. [ ] **未来视角** — 3-5 年后自己要创业 / 带团队，需要提前掌握作为创始人 / CEO 应该懂的股权设计、融资、退出全套机制

调研对象的具体处境（背景信息，不需要在报告里直接讨论）：

- 境内主体的公司，赛道是 toC 创作者工具 + 社群
- 公司没融过资 / 自有资金阶段
- 拿到的是创始人 CEO 口头承诺的"股份"，未签订任何文件
- 早期核心员工（不是联创），全职 + 市场工资 + 股份是额外激励

# Part 2 · 股权 vs 期权 vs RSU vs 虚拟股 vs 分红权

## 2.1 五种工具的对比矩阵（核心表格）

## 2.2 期权的全套机制（这是员工最常拿到的形式，必须吃透）

- **Q2.2.1 期权池（ESOP）从哪里来**：什么时候建、谁稀释（创始人稀释还是融资时投资人共同稀释）、典型规模 10-20%
- **Q2.2.2 授予（Grant）**：授予协议（Grant Letter）里通常写什么、授予日 vs 生效日
- **Q2.2.3 行权价（Strike Price / Exercise Price）**：怎么定（FMV 公允价值）、为什么早期员工的行权价低、A 轮后入职的员工为什么行权价会变高
- **Q2.2.4 Vesting Schedule（兑现时间表）**：典型 "4 年 + 1 年 Cliff" 是什么意思、月度 vesting vs 季度 vesting、Back-loaded vs Front-loaded
- **Q2.2.5 Cliff（悬崖期）**：1 年内离职 = 一份都拿不到 — 业内通行做法
- **Q2.2.6 行权（Exercise）**：什么时候能行权、用什么钱行权、行权后变成什么（普通股 / 还是要等流动性事件）
- **Q2.2.7 行权窗口（Exercise Window）**：离职后通常 90 天必须行权，否则作废 — 这是巨坑（介绍业内推动的"10 年行权窗口"改革，Pinterest / Quora / Coinbase 等）
- **Q2.2.8 加速兑现（Accelerated Vesting）**：单触发（Single-trigger）vs 双触发（Double-trigger），通常在被并购时触发

# Part 7 · 给"未来创始人 / CEO"的反视角

## 7.1 员工股权 vs 联创股权的差异详解（独立小节，核心内容）

用一张大表对比 8 个维度：

| 维度             | 联创股权（Co-founder Equity）                            | 员工期权（Employee Option）          |
| ---------------- | -------------------------------------------------------- | ------------------------------------ |
| 法律形式         | 通常直接持股 / 持股平台实股                              | 期权（行权前不是股东）               |
| 比例             | 通常 5-50%                                               | 通常 0.05-2%                         |
| 是否承担创业风险 | 是（低工资 / 无工资 / 现金垫付）                         | 否（市场工资）                       |
| Vesting 是否适用 | 创始人也要 vesting（业内最佳实践，但很多中国创业者不做） | 强制 vesting                         |
| 投票权 / 董事会  | 通常有                                                   | 通常无                               |
| 离职处理         | 复杂（回购权 / Drag-along / 竞业限制）                   | 简单（已 vest 部分 + 90 天行权窗口） |
| 知情权           | 法定查账权                                               | 通常没有                             |
| 心理预期         | 公司是自己的                                             | 公司是创始人的，自己是高级雇员       |

- **Q7.1.1 灰色地带**：早期联合创始人（很早期加入但不算法律意义上的联创）的处境 — 通常给"超额期权"或"小份额实股"

# Part 8 · 持有股份能干什么（深问题）

## 8.1 法律意义上的权利

- **Q8.1.1 投票权**（股东大会决议、累积投票制）
- **Q8.1.2 查账权 / 知情权**（《公司法》原第 33 条 / 新法第 57 条）：股东可以查公司财务报表、会计账簿、会议决议 — 重点讲实操中怎么行使

## 8.2 财务意义上的能力

- **Q8.2.1** 抵押贷款：未上市公司股权能不能抵押 — 法律上可以但实操几乎没人接受
