# Analysis

本目录只保存基于 `../official/google-design-md/` 的本地拆解与验证记录。

## 文件

- `official-design-md-breakdown.md`：对 Google 官方 DESIGN.md 规范和 examples 的拆解入口。
- `design-system-contract-deep-dive.md`：按结构、思想、token、dark mode、组件、工程化、隐藏问题、专家追问八层框架做深度拆解。
- `daily-development-usage.md`：把 DESIGN.md 放进日常 Codex / Cursor 项目的建议结构。
- `sources.json`：来源、commit、release、raw file URL、本地路径和尺寸台账。
- `verification.md`：本地校验命令与结果。

## 边界

- 不存放上游原始文件。
- 不修改 `../official/` 中的官方快照内容。
